# Portafolio - Ciencia de Datos e Inteligencia Artificial

Bienvenido/a a mi portafolio de proyectos desarrollados durante la Tecnicatura en Ciencia de Datos e Inteligencia Artificial.  
En este repositorio reúno trabajos prácticos, análisis exploratorios y modelos aplicados a distintos problemas de datos, estadística y aprendizaje automático.

## Sobre mí

Soy estudiante de la Tecnicatura en Ciencia de Datos e Inteligencia Artificial, con interés en análisis de datos, estadística, machine learning y automatización.  
Este portafolio reúne algunos de los trabajos que fui desarrollando a lo largo de mi formación académica, aplicando herramientas de análisis, visualización y modelado de datos en Python.

## Tecnologías utilizadas

- Python
- Jupyter Notebook
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

## Proyectos incluidos

### 1. Análisis Exploratorio de Siniestros Viales en Argentina
Proyecto de análisis exploratorio de datos orientado a identificar patrones, distribuciones y relaciones en información sobre siniestros viales en Argentina.  
Incluye limpieza de datos, análisis descriptivo y visualizaciones para interpretar los principales hallazgos (Proceso KDD).

### 2. Análisis PCA
Aplicación de Análisis de Componentes Principales (PCA) para reducción de dimensionalidad e interpretación de variables.  
Permite explorar cómo resumir la información contenida en múltiples variables conservando la mayor cantidad posible de varianza.

### 3. ANOVA y Mutual Information
Trabajo enfocado en el análisis de relación entre variables y evaluación de su relevancia respecto de una variable objetivo.  
Incluye técnicas estadísticas y de selección de variables para apoyar la interpretación de los datos.

### 4. Estimador Chi-Cuadrado
Análisis basado en el test Chi-cuadrado para estudiar asociación entre variables categóricas.  
Incluye planteo de hipótesis, cálculo del estadístico e interpretación de resultados.

### 5. Reconocimiento Automatizado de Emociones
Proyecto orientado a la aplicación de técnicas de inteligencia artificial para clasificación, enfocado en el reconocimiento automatizado de emociones.

## Objetivo del repositorio

Este repositorio fue creado con el fin de centralizar y mostrar proyectos académicos que reflejan mi proceso de formación en ciencia de datos e inteligencia artificial.  
Cada notebook representa una instancia de aprendizaje práctico en la que trabajé con problemas de análisis, modelado e interpretación de datos.

## Cómo explorar este repositorio

Los proyectos se encuentran en formato Jupyter Notebook (`.ipynb`).
